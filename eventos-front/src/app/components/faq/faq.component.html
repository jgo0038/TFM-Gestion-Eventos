Preguntas frecuentes

¿Es necesario pagar por algún servicio?

No, la aplicación ofrece todas sus posibilidades completamente gratis al haber creado una cuenta.

¿Cómo se puede añadir una nueva ciudad?

Para la creación de una nueva ciudad en la que comenzar a añadir eventos, hay que ponerse en contacto con el soporte/administrador, el cual se encargará de crear la ciudad lo antes posible y quedará creada para siempre.

¿Cómo se puede añadir una nueva categoría?

Para la creación de una nueva categoría, hay que ponerse en contacto con el soporte/administrador, el cual se encargará de crear la ciudad lo antes posible y quedará creada para siempre.

¿Es obligatorio inscribirse para acudir a la actividad?

No, la inscripción es una información adicional que se le ofrece al creador para llevar un recuento a grandes rasgos de la gente que acudirá y guardar el evento en el perfil del usuario inscrito. Si se desean comprar entradas o hacer una reserva completa se ofrece el contacto del creador para tramitarlo.

¿Se pueden realizar compras a través de la web?

No, las compras las gestiona directamente el anunciante, de momento...

¿Para realizar una inscripción a una actividad/evento es necesario estar registrado?

Sí, la acción para inscribirse es única para usuarios registrados.

¿Para crear una actividad/evento es necesario estar registrado?

Sí, la acción para inscribirse es única para usuarios registrados.

¿Cómo puedo editar/borrar una actividad/evento creada por mi?

Las actividades/eventos creadas aparecen en la lista del perfil de cada uno, desde ahí se puede acceder al borrado o edición de cada uno de ellos.

¿Puedo hacer que un evento vuelva a ser visible una vez se haya pasado su fecha?

Sí, desde tu perfil puedes acceder a la edición del evento y actualizar la fecha a una posterior, de forma que volverá a mostrarse en el listado de eventos futuros con toda la información igual y manteniendo los comentarios.
